# Learning AI

Proyectos y ejercicios para aprender IA y machine learning.

## Setup

```bash
make i
```

## Run

```bash
make run file=perceptron_mse.py
make run file=perceptron_gd.py
```

## Estructura

- `perceptron_mse.py` - Perceptrón con solución cerrada (least squares)
- `perceptron_gd.py` - Perceptrón con gradient descent
- `Makefile` - Comandos de build
- `requirements.txt` - Dependencias Python
